CRM–ERP Data Reliability Audit

A full reconciliation pipeline that pulls deal data from both RD Station CRM and TOTVS ERP, matches records by deal number, and flags any discrepancy in deal value or status. Results are written back to Google Sheets with clear labels: "OK", "DIVERGE", or "Not Found".

Handles Brazilian number formatting (1.234,56), token rotation for RD Station OAuth2, automatic Heroku config-var updates, and batch Google Sheets writes for performance. Built to run on a schedule and surface data integrity issues before they become business problems.

  store = _load_store()
  token = store.get("refresh_token")
  if new_refresh != token:
